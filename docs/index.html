The Cat Gallery

This is a practice project I challenged myself with to improve my work efficiency, and my skills with HaxeFlixel, reactive UI, and API requests.

What it does

The cat gallery stores and displays photos and information about cats. Photos can be downloaded from TheCatAPI.

How to Use it

Because this app uses an API with limited requests, I cannot distribute an executable version.

However, I can distribute the source code and building instructions on github.com
